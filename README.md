# Credit_Card_Analysis
This repository contains synthetic credit card transaction data used for exploratory data analysis, fraud detection Modeling, and dashboard visualization. It showcases techniques in data cleaning.

Project Objective
- Analyze credit card transaction behavior to identify trends and potential fraud indicators.
- Build interactive dashboards to visualize customer spending, merchant categories, and transaction risk.
- Apply data cleaning, transformation, and visualization techniques to derive actionable insights.

Project Steps
- Data Cleaning
- Removed nulls, duplicates, and outliers
- Standardized date/time formats and currency values
- Exploratory Data Analysis (EDA)
- Identified top spending categories
- Created new fields like Week-on-Week Revenue, Age-Group etc
- Dashboard Development
- Built slicers for Quarter, Gender, and Week-Start-Date
- Designed KPI cards for Total Revenue, Total Interest, Transaction Amount and Transaction Count
- Used Stacked Bar Chart, Line Chart, and Treemap for visual storytelling

Tools Used
- Power BI: Interactive dashboard creation
- Excel: Initial data exploration and validation
